# Architecture Overview

This project is an internal process audit system built using:

- Power Apps (Canvas App)
- SharePoint (Data Source)
- Power Automate (Automation)
- Power BI (Analytics)

## Layers

### 1. Data Layer
- SharePoint Lists
- Structured and normalized data

### 2. Application Layer
- Power Apps Canvas App
- Business logic and UI

### 3. Automation Layer
- Power Automate flows
- PDF generation and notifications

### 4. Analytics Layer
- Power BI dashboards
